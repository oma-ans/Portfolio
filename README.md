## Getting Started

### Prerequisites

No special requirements! Just a modern web browser.

### Installation

1. Clone or download this repository
2. Open `index.html` in your web browser
3. That's it! No build process or dependencies needed.

### Usage

- **Mouse Navigation**: Click menu buttons to navigate between sections
- **Keyboard Shortcuts**:
  - `1` - About
  - `2` - Education
  - `3` - Experience
  - `4` - Leadership
  - `5` - Research
  - `ESC` - Return to title screen
